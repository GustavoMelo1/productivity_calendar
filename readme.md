CALENDARIO DA PRODUTIVIDADE

Projeto relampago para resolver um problema pessoal de organização. Eu precisava de um painel de produtividade que ficasse o tempo todo no meu OLHO, e ferramentas como o Notion ou o Trello não entregavam o que eu queria. Como não encontrei nada pronto que me ajuda-se, decidi fazer um projeto relampago para colocar de WALLPAPER na minha area de trabalho.

O sistema não tem servidor, guarda tudo diretamente no LocalStorage do browser e roda na minha area de trabalho (Wallpaper Engine).
